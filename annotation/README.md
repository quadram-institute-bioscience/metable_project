* contigs2sql.pl

```
Usage: 
contigs2sql.pl contigs_fasta [prefix] [project_token] > contigs.sql
  - prefix:         table names prefix (default: $default_prefix)
  - project_token:  a string without spaces in the ProjectID,ProjectOwner format (default: $default_token)
```

prokka2sql.pl annotation.gff > genes.sql
